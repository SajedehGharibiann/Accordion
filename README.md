This is a simple and responsive FAQ Accordion component built using HTML, CSS, and Vanilla JavaScript.
The project allows users to click on a question to expand the answer while automatically closing other open items. It demonstrates DOM manipulation, event handling, CSS transitions, and basic UI interaction logic.
